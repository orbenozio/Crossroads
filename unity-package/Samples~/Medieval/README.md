# Medieval skin

A ready-made `Theme` that dresses the engine's neutral default in the original medieval look (bronze accents,
warm-gold choice hints, engraved bronze plaque edges, antique-bronze meter icons).

Import it via the Package Manager (Crossroads Engine -> Samples -> Medieval), or copy `Medieval.asset` into
your game and point your `GameBootstrap.theme` at it - then override from there. It sets only color tokens, so
it composes with any art/audio/typography you add.

See `../../THEMING.md` for the full token model and the exact recipe this asset encodes.
